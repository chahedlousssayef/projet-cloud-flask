Personnaliser les chiffres

Clique sur un chiffre pour uploader une image. Les images sont stockées sur Azure Blob Storage.

API Endpoints

L'application expose une API REST pour gérer les images des chiffres et les items CRUD.

GET
/api/clock/digits Liste les chiffres avec image
POST
/api/clock/digits/{digit} Upload une image pour un chiffre
GET
/api/clock/digits/{digit}/image Récupère l'image d'un chiffre
DEL
/api/clock/digits/{digit} Supprime l'image d'un chiffre
GET
/api/items Liste tous les items
POST
/api/items Crée un nouvel item
